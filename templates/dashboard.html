Your Action
Open each exported batch file, paste the contents into Claude Code or Claude Chat, and save the JSON response. This is the core analysis step — Claude reads each passage and extracts obligations, definitions, thresholds, and ambiguities.
How to do this:
  1. Open a file from the export/ folder (e.g., batch_001.txt)
  2. Copy all the text in that file
  3. Paste it into Claude Code or Claude Chat
  4. Wait for Claude's response (a JSON array)
  5. Save the JSON response as export/batch_001_results.json
  6. Repeat for each batch file
